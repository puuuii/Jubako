# Hierarchical Clipboard

階層化されたクリップボード管理は、本プロジェクトの最もユニークな機能です。

## 構造
- **ルート**: 全てのアイテムとフォルダを保持。
- **フォルダ**: アイテムおよび他のフォルダを含むコンテナ。
- **アイテム**: クリップボードの内容（テキスト、画像、RTFなど）。

## 操作
1. **履歴から永続化**: 通常の履歴一覧から特定のアイテムを右クリックし、「永続化」を選択してフォルダを指定。
2. **階層移動**: UI 内でフォルダを作成し、アイテムを自由に移動可能。
3. **リネーム**: 永続化したアイテムには表示名（ラベル）を設定可能。
