# Specifications

Jubako の主な仕様は以下の通りです。

## 基本機能
Windows 標準のクリップボード履歴 (`Win + V`) と同等の履歴機能を備えます。
- コピーしたテキスト、画像（将来的にはファイルなど）を自動で保存。
- 直近の履歴を一覧表示。

## 階層化永続化
履歴として保存されたアイテムを、ユーザーがフォルダ分けして永続的に保存できます。
- アイテムをドラッグ＆ドロップまたはコンテキストメニューから階層構造（フォルダ）に移動。
- 階層化されたアイテムはアプリを再起動しても保持されます。
- 階層構造はネスト可能です。

## UI レイアウト
1.  **検索窓**: 最上部に配置し、履歴と永続化アイテムの両方をリアルタイムフィルタリング。
2.  **2カラム構成**:
    *   **左ペイン (Sidebar)**: 階層構造（フォルダ）を表示。ドラッグ＆ドロップでフォルダの並べ替えやネストが可能。
    *   **右ペイン (Main)**: 選択したフォルダ内のアイテム、または「履歴（History）」を一覧表示。
3.  **コンテキストメニュー**: アイテムを右クリックして「永続化（フォルダへ移動）」「名前変更」「削除」が可能。

## ウィンドウの振る舞い
- `Win + Alt + V` でマウスカーソル位置にポップアップ。
- タスクバーには表示しない（ツールウィンドウ）。
- ウィンドウ外をクリック（フォーカス喪失）した際に自動的に閉じる。
- アイテムを選択（Enter または Click）すると、クリップボードにコピーした上でウィンドウを閉じる。
